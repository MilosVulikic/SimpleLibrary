# SimpleLibrary
Simple Math Library

This library will contain simle operations such as: 
plus, minus, divide, multiply, max value, min value, median.
All of these operations can be done on the following types of variables :
int, float and double
, as well as on the array of the same type.

<p>NameOfTheFunction:</p>
<p>Plus</p>
<p>Minus,</p>
<p>Divide,</p>
<p>Multiply,</p>
<p>Max,</p>
<p>Mix,</p>
<p>Median</p>
<br>
<p>int NameOfTheFunction(int value1, int value2) - Input is integer, output is Integer</p>
<p>int NameOfTheFunction(float value1, float value2) - Input is float, output is float</p>
<p>int NameOfTheFunction(double value1, double value2) - Input is double, output is double</p>
<p>int[] NameOfTheFunction(int[] value1, int[] value2) - Input is array of integers, output is array of integers</p>
<p>float[] NameOfTheFunction(float[] value1, float[] value2) - Input is array of floats, output is array of floats</p>
<p>double[] NameOfTheFunction(double[] value1, double[] value2) - Input is array of doubles, output is array of doubles</p>
